---
दो दस्तावेज़ों को एक ही जगह स्क्रॉल करके रखना आमतौर पर बार-बार टैब बदलने की झंझट बन जाता है। यह extension आपके browser tabs को आपस में जोड़ता है — एक tab में scroll करें, और सभी connected tabs तुरंत उसी position पर पहुँच जाते हैं।
---

यह कैसे काम करता है

1. Extension icon पर click करें और tab selector खोलें
2. जिन tabs को sync करना है उन्हें search करके select करें
3. "Start Sync" पर click करें — बस, हो गया

जब कई tabs पर एक ही URL खुली हो, तो एक smart notification एक ही click में उन्हें sync करने का विकल्प देती है।

---

मुख्य विशेषताएँ
• किसी भी संख्या में tabs के बीच real-time scroll synchronization
• Command palette के साथ search — title या URL से tabs ढूँढें, domain के अनुसार filter करें
• Manual position adjustment — किसी एक tab की alignment ठीक करने के लिए Mac पर Option या Windows पर Alt दबाए रखें
• Duplicate URLs मिलने पर smart auto-sync सुझाव
• URL navigation sync — एक tab में link click करें और सभी linked tabs उसे follow करें
• Domain exclusion — किसी खास site के लिए सुझाव हमेशा के लिए बंद करें
• Sleep या interruption के बाद auto-reconnection, कोई manual action नहीं चाहिए
• हर काम के लिए पूरे keyboard shortcuts

---

यह किनके लिए है?
• अनुवादक जो मूल और अनुवादित text की तुलना करते हैं
• Developers जो अलग-अलग versions या files में code changes देखते हैं
• Researchers जो एक साथ कई papers पढ़ते हैं
• Students जो lecture notes और reference material साथ-साथ compare करते हैं
• कोई भी जो एक साथ दो या अधिक दस्तावेज़ों पर काम करता है

---

जहाँ Sync उपलब्ध नहीं है
Browser की security restrictions की वजह से कुछ pages पर scroll sync काम नहीं करता:
• Google services (Docs, Drive, Gmail, Sheets, Slides)
• Web apps जैसे Figma, JIRA, Notion, और Microsoft Office Online
• Browser के internal pages (chrome://, edge://, about:)
• Extension stores और search engine result pages
• PDF files और viewers
• Login और authentication pages
• Special URLs (file://, view-source:, data:)

Restricted tabs, selection list में disabled दिखते हैं और एक tooltip उस सीमा की जानकारी देता है।
