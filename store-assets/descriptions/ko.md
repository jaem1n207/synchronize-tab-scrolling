---
두 문서를 같은 위치에 맞춰 읽으려면 탭을 번갈아 가며 스크롤해야 합니다. 이 확장 프로그램은 브라우저 탭을 연결하여, 한 탭에서 스크롤하면 연결된 모든 탭이 같은 위치로 자동 이동합니다.
---

사용 방법

1. 확장 프로그램 아이콘을 클릭하여 탭 선택 화면을 엽니다
2. 제목이나 URL로 탭을 검색하고 동기화할 탭을 선택합니다
3. "동기화 시작" 버튼을 클릭하면 즉시 적용됩니다

동일한 URL을 가진 탭이 여러 개 열려 있으면 자동으로 동기화를 제안하는 알림이 표시됩니다.

---

주요 기능
• 여러 탭의 스크롤을 실시간으로 동기화
• 커맨드 팔레트 형태의 탭 선택 — 제목/URL 검색, 도메인 필터링 지원
• 수동 위치 조정 — Option(Mac) 또는 Alt(Windows)를 누른 채 스크롤하여 개별 탭 위치를 미세 조정
• 동일 URL 탭 감지 시 자동 동기화 제안
• URL 내비게이션 동기화 — 한 탭에서 링크를 클릭하면 연결된 탭도 함께 이동
• 도메인 제외 관리 — 특정 사이트의 동기화 제안을 영구적으로 차단
• 절전 모드나 연결 중단 후 자동 재연결
• 모든 주요 동작에 키보드 단축키 지원

---

이런 분들에게 유용합니다
• 원문과 번역문을 나란히 비교하는 번역가
• 코드 변경 사항을 여러 파일에서 검토하는 개발자
• 여러 논문을 동시에 읽는 연구자
• 강의 자료와 참고 문헌을 함께 확인하는 학생
• 두 개 이상의 문서를 나란히 놓고 작업하는 모든 사용자

---

지원되지 않는 페이지
브라우저 보안 정책 또는 사이트 제한으로 인해 다음 페이지에서는 사용할 수 없습니다:
• Google 서비스 (Docs, Drive, Gmail, Sheets, Slides)
• Figma, JIRA, Notion, Microsoft Office Online 등 웹 애플리케이션
• 브라우저 내부 페이지 (chrome://, edge://, about:)
• 확장 프로그램 스토어 및 검색 결과 페이지
• PDF 파일 및 PDF 뷰어
• 로그인 및 인증 페이지
• file://, view-source:, data: 등 특수 URL

해당 탭은 선택 목록에서 비활성 상태로 표시되며, 제한 사유가 툴팁으로 안내됩니다.
