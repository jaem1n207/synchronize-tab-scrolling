Section 1: Top of Page

This is a test page for the Synchronize Tab Scrolling extension.

Open this page in multiple tabs and use the extension popup to sync their scroll positions.

How to test:
  1. Open this file in 2-3 browser tabs
  2. Click the extension icon in your browser toolbar
  3. Select the tabs you want to sync
  4. Click "Start Sync"
  5. Scroll in any tab and watch the others follow!

Section 2: Features

The extension supports two sync modes:

  • Ratio-based sync: Syncs by scroll percentage
  • Element-based sync: Syncs by matching DOM elements

You can also enable URL synchronization to navigate tabs together.

Section 3: Middle Content

This section is in the middle of the page.

When you scroll here, other synced tabs will scroll to the same relative position.

The extension uses a 200ms debounce to ensure smooth scrolling.

Section 4: Testing Area

Try different scroll speeds and patterns:

  • Slow scrolling
  • Fast scrolling
  • Page Up/Down keys
  • Home/End keys
  • Clicking scroll bar

All methods should sync properly across tabs.

Section 5: Advanced Features

The extension includes:

  • Multi-language support (6 languages)
  • Accessibility features (WCAG 2.1 AA)
  • Draggable control panel in each tab
  • Keyboard shortcuts support
  • High contrast mode

Troubleshooting:
  • If tabs aren't syncing, check that they're selected in the popup
  • Some restricted pages (like chrome:// URLs) cannot be synced
  • Refresh the extension if sync stops working
